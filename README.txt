# Getting Started

### Reference Documentation

Requirment: Mac, JDK 8, gradle, Maven Postman

Set "API_KEY" to run WeatherApp
export API_KEY="API_KEY_WITH_FROM_OPEN_WEATHER_MAP_ORG_API"

Compile: ./gradlew clean build
Running server: ./gradlew bootrun



##  Swagger Documentation:
http://localhost:8080/weatherapp/swagger-ui/index.html
